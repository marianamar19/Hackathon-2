# 📒 Agenda Telefónica 

## Descripción
Sistema de gestión de agenda telefónica en Java. Permite añadir, listar, buscar y eliminar contactos.  
La agenda se puede crear con un tamaño definido por el usuario o con un tamaño por defecto de 10.


## Requerimientos
1. Trabajar bajo poo.
2. Organizar proyectos en packages.
3. Crear metodos para Manager y Developer crear, recuperar, buscar, eliminar en la DB.

**🛠️ Funcionalidades**
- **Añadir contacto**: No permite duplicados (mismo nombre).
- **Existe contacto**: Verifica si un contacto está en la agenda.
- **Listar contactos**: Muestra todos los contactos registrados.
- **Buscar contacto**: Localiza un contacto por nombre y muestra su teléfono.
- **Eliminar contacto**: Borra un contacto existente.
- **Agenda llena**: Indica si se alcanzó el límite de contactos.
- **Espacios libres**: Muestra cuántos contactos más se pueden ingresar.


**Organización de Packages**
- *app*: `Main.java`
- *model*: `contacto.java`, 
- *service*: `Agenda.java`.
